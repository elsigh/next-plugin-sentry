# Next.js + Sentry

Capture unhandled exceptions with Sentry in your Next.js project

## Installation

```
npm install kamilogorek/next-plugin-sentry
```

or

```
yarn add kamilogorek/next-plugin-sentry
```

### Usage

Create a next.config.js

```js
module.exports = {
  experimental: { plugins: true }
}
```
